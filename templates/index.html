Exoplanet Microservice API Documentation

1. Add an Exoplanet

Endpoint: /exoplanets (POST)

Request Body:

{
    "id": "string",
    "name": "string",
    "description": "string",
    "distance": float64,
    "radius": float64,
    "mass": float64 (optional),
    "type": "GasGiant" | "Terrestrial"
}

2. List Exoplanets

Endpoint: /exoplanets (GET)

3. Get Exoplanet by ID

Endpoint: /exoplanets/{id} (GET)

4. Update Exoplanet

Endpoint: /exoplanets/{id} (PUT)

Request Body:

{
    "name": "string",
    "description": "string",
    "distance": float64,
    "radius": float64,
    "mass": float64 (optional),
    "type": "GasGiant" | "Terrestrial"
}

5. Delete Exoplanet

Endpoint: /exoplanets/{id} (DELETE)

6. Fuel Estimation

Endpoint: /exoplanets/{id}/estimate-fuel (GET)

Query Parameter: crew_capacity (int)
